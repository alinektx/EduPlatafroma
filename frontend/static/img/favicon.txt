Para criar o favicon.ico:

1. Use o arquivo favicon.svg como base
2. Converta para ICO usando:
   - Online: https://convertio.co/svg-ico/
   - Ou use ferramentas como ImageMagick
   - Tamanho recomendado: 32x32 pixels

3. Salve como: favicon.ico na pasta static/img/

   <link rel="icon" type="image/x-icon" href="/static/img/favicon.ico">
